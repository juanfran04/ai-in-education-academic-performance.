# Impact of Generative AI Usage on Academic Performance

Analysis of the relationship between Generative AI usage and academic performance.

## Overview

This project investigates the relationship between Generative AI usage and academic performance among university students.

## Objectives

- Analyze the relationship between AI usage and GPA.
- Evaluate the predictive contribution of AI-related variables.
- Compare traditional academic predictors with AI-related factors.

## Dataset

**Source:** Kaggle

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statsmodels
- Scikit-Learn

## Methodology

1. Data Preparation
2. Exploratory Data Analysis
3. Hierarchical Regression
4. Statistical Inference
5. Random Forest

## Key Findings

- Previous GPA was the strongest predictor of academic performance.
- Study hours maintained a significant positive association with GPA.
- Weekly AI usage showed a statistically significant but limited practical effect.
- The quality and purpose of AI usage were more strongly associated with academic outcomes than usage time alone.

## Limitations

- Observational dataset.
- Self-reported variables.
- Potential omitted variables.
